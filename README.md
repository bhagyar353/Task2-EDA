# 📊 Sales Data Exploratory Data Analysis

This project performs Exploratory Data Analysis (EDA) on sales data to uncover meaningful insights, trends, and patterns that can support data-driven business decisions.

---

## 🔍 Project Objective
The objective of this project is to analyze sales data and identify key insights related to sales performance, customer behavior, and product trends.

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Google Colab  

---

## ⚙️ Steps Performed
- Data Loading  
- Data Cleaning and Preprocessing  
- Handling Missing Values  
- Exploratory Data Analysis (EDA)  
- Statistical Summary  
- Data Visualization  

---

## 📈 Key Insights
- Identified top-performing products based on sales  
- Observed sales trends over time  
- Analyzed customer purchasing patterns  
- Found insights that can help improve business decisions
